# 个人网站

这是我的个人网站，展示我的个人信息、技能和作品。

## 功能特性

- 个人信息展示
- 技能展示
- 作品展示
- 联系方式
- 后台管理系统

## 技术栈

- HTML
- CSS
- JavaScript
- 本地存储（LocalStorage）

## 如何使用

1. 直接打开 `index.html` 文件查看个人网站
2. 打开 `admin/index.html` 进入后台管理系统
3. 在后台可以上传头像和关于我的照片
4. 可以管理个人信息和作品

## 项目结构

```
.
├── admin/          # 后台管理系统
├── css/            # 样式文件
├── images/         # 图片资源
├── js/             # JavaScript 文件
├── index.html      # 首页
└── README.md       # 项目说明
```
